RESOURCES 
https://www.robinwieruch.de/minimal-react-webpack-babel-setup/

After countless terrible articles this link provided the basis for this simple react development environemnt.

OVERVIEW

bundler: webpack
js converter: babel
file serving: webpack dev server
hot loading: react hot loader


USAGE

-- run start

NOTES

-- Compared to the compile time of a big project this is almost instant, expecially with hot loader. this is a great place to develop new components in a vaccuum.
